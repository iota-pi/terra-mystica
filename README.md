# Terra Mystica

## Getting started

Open a bash shell and run:

```
./ci.sh build
```

Then, if using VS Code, use the reopen in container command.

To test it, run:

```
docker compose run --rm app
```

## Running tests

To run the tests locally, use:

```
./ci.sh test
```

To auto-format your code, use:

```
./ci.sh format
```
